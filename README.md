# Риа Олабеага · Приливы

Небольшое PWA-приложение для просмотра уровня воды, приливов и отливов в риа Олабеага, Бильбао.

Открыть приложение: https://aerotsunami.github.io/olabeaga_waves/

## Что показывает

- текущий уровень воды в метрах;
- направление движения воды: прибывает, уходит или разворачивается;
- скорость изменения уровня в сантиметрах в час;
- динамику за 1, 3 и 6 часов;
- график уровня на ближайшие дни;
- ближайшие приливы и отливы.

## Данные

Приложение использует Open-Meteo Marine API и запрашивает параметр `sea_level_height_msl` для координат Олабеаги.

Данные показываются в местном времени Бильбао. Обновление происходит примерно раз в 30 минут.

## Как устроено

Это статический сайт без сборки и серверной части:

- `index.html` — интерфейс, график и логика расчета;
- `manifest.json` — настройки PWA;
- `sw.js` — service worker для кэша оболочки приложения;
- `icon-*.png` — иконки приложения.

## Локальный запуск

Достаточно открыть `index.html` в браузере. Для полноценной проверки PWA и service worker лучше запустить папку как локальный статический сервер, например:

```bash
python -m http.server 8000
```

После этого открыть `http://localhost:8000`.
